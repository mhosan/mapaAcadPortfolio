  • Visor gis Web de mapas (Desarrollo personal):
    • Permite seleccionar, en la Pcia. de Bs. As, Partidos, escuelas y circuitos electorales.
    • Contiene un geolocalizador con dos opciones:
      1. Local, con la Api de la Secretaria de Modernización dependiente de la Presidencia de la Nación Argentina (https://datos.gob.ar/apis), es la primer opción, aparece como "S.Modern.".
      2. Global con la libreria LocationIQ: https://locationiq.com/. Es la segunda opción, aparece como "LocationIq".
    • Incluye gráficos COVID-19 (Chartjs). Hecho con Angular 9, Leaflet 1.4.0 y Bootstrap 4.4
    • Hecho con Angular 9, Leaflet 1.4.0 y Bootstrap 4.4.
    • Persiste datos en Firebase. Es una aplicación web con backend como servicio (BaaS): Firebase proporciona servicios backend preconfigurados y escalables.
    • Se usó mapbox/leaflet-omnivore para que Leaflet admitiera los formatos CSV, GPX, KML, WKT y TopoJson.
    • Url: https://argengis.web.app
