Pronostico del tiempo

Probar un MCP que provee el pronostico del tiempo, en formato JSON, luego el LLM lo formatea a lenguaje natural.



Ingresar una ciudad

Se envia la información (ciudad) a un backend (https://serverminimo.onrender.com/api/weather) el cual toma el dato de la ciudad y llama a un MCP que provee el pronostico del tiempo en formato JSON. Y luego envia el JSON a un LLM que lo formatea a lenguaje natural. El servidor MCP está en Vercel: https://mcpserver-hazel.vercel.app/. El repositorio del MCP está en GitHub:https://github.com/mhosan/mcpserver
